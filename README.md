# POE-website-M-Sweets-Bakery
Website Name: M Sweet Treats Bakery

Description:
A fully interactive bakery website showcasing products, services, pricing, and contact information. Includes dynamic slideshows, calculators, order forms, and a live footer timestamp.

Features:
- Homepage slideshow with transitions and controls.
- Footer shows real-time last updated timestamp.
- Price calculator, order form, and downloadable PDF.
- Styled info boxes on About, Services, Contact pages.
- Product cards with hover effects.
- Centralized 'script.js' for all JavaScript.

Pages:
1. Home – Slideshow and info box.
2. About – Bakery history and founder details.
3. Services – List of bakery services.
4. Products – Product showcase.
5. Prices – Price list, calculator, PDF download, order form.
6. Contact – Store location and contact form.

How to Use:
- Open 'index.html' to start.
- Navigate via the menu.
- Use slideshow controls and forms as needed.

Technical Info:
- All JS in 'script.js'.
- Styles in 'style.css' and 'css2.css'.
- Images in 'images/' folder.
- Mobile responsive and HTML5 compliant.
